# GA-deep-scroll
This script sends an event about deepness of scroll to Google Analytics

This script sends the event with category: "Scroll " and action:
- from 0% to 20%
- from 20% to 40%
- from 40% to 60%
- from 60% to 80%
- from 80% to 100%

when visitor scrolls down the page. And this event is nonInteraction.
